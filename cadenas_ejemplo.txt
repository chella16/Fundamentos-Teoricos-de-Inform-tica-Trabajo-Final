# Archivo de ejemplo para el reconocedor de gramática
# Las líneas que comienzan con # son comentarios y se ignoran
# Las líneas vacías también se ignoran

# Ejemplos válidos
comienza  termina
comienza leer <x> termina
comienza escribir <mensaje>; termina
comienza <resultado>=<suma>; termina
comienza si <condicion> entonces <accion> finsi termina
comienza si <condicion> entonces <accion> sino <accion> finsi termina
comienza mientras <contador> hacer <incremento> finmientras termina
comienza repetir <calculo> hastaque <limite> termina

# Ejemplos con estructuras anidadas
comienza si <condicion> entonces mientras <cond2> hacer <inst> finmientras finsi termina
comienza mientras <condicion> hacer si <cond2> entonces <inst> finsi finmientras termina
comienza mientras <condicion> hacer si <condicion> entonces <accion> sino <accion> finsi finmientras termina

# Ejemplos con PascalCase sin espacios
ComenzaLeer<x>Termina
ComenzaEscribir<mensaje>;Termina
Comienza<resultado>=<suma>;Termina
ComenzaSi<condicion>Entonces<accion>FinSiTermina
ComenzaMientras<contador>Hacer<incremento>FinMientrasTermina
ComenzaRepetir<calculo>Hasta<limite>Termina

# Ejemplos inválidos (para pruebas)  
leer <x> termina
comienza leer termina
comienza si <cond> entonces <inst> termina
comienza escribir <mensaje> termina
comienza <resultado>=<suma> termina
